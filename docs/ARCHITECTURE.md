### Overview {#overview-en}

The Enterprise Setup Repository implements a modern, scalable CI/CD infrastructure with multiple integrated quality, security, and automation layers.

#### Architecture Principles

**1. Defense in Depth**
- Multiple security layers (CodeQL, Snyk, TruffleHog, Gitleaks)
- Redundant validation at different levels
- Fail-safe mechanisms

**2. Shift-Left Security**
- Security checks at commit time (Git Hooks)
- Pre-merge validation through Branch Protection
- Continuous security scanning

**3. Automated Quality Gates**
- Automated AI code reviews (CodeRabbit, CodiumAI)
- Quality metrics (Codacy, CodeFactor)
- Test coverage enforcement (Codecov)

**4. Infrastructure as Code**
- Workflows as code (YAML)
- Branch protection as code (JSON)
- MCP server configuration as code

**5. Observability**
- Continuous monitoring (MCP Health Check)
- Drift detection
- Workflow logs and reports
